# Prostate Mortality version AI
### Prostate Cancer Social determinants & Clinicopathological Unified Model

This repository contains the deployment code for the model as described in the manuscript submitted.

**Model Overview:**
- **Architecture:** DeepHit Multi-task Neural Network
- **Data Source:** SEER Custom Database (2010-2020), N=224,573
- **Primary Aim:** Integration of Social Determinants of Health (SDoH) into Competing Risk Mortality Prediction.

**Compliance:**
This tool is for **Research Use Only**. It incorporates SHAP-based transparency to comply with guidelines.
